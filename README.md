This project showcases a real-world data cleaning pipeline built using Python and Pandas.The dataset represents banking customer information
and includes common data quality issues found in real datasets, such as missing values, duplicate records, inconsistent formatting, and extreme outliers.

The project demonstrates practical data analyst skills including data exploration, missing value handling, duplicate detection, text standardization,
feature engineering, and outlier detection using the IQR method.

After applying these cleaning steps, the dataset is transformed into a structured and reliable dataset ready for analysis, visualization, and machine learning tasks.
